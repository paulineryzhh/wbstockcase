Анализ товарных запасов WB: pandas + Power BI, поиск неликвида и рисков out-of-stock

# Анализ запасов и приоритетов пополнения на маркетплейса Wildberries 

## Задача
Кабинет продавца на Wildberries: 2470 уникальных товаров на 37 складах.
Нужно понять: где не хватает товара при живом спросе (риск упущенных
продаж), и где деньги заморожены в неликвиде — товаре, который лежит
на складе, но не продаётся. Также оценить, какие товары и бренда приносят наибольшую
выручку

## Данные
Отчёт по продажам из личного кабинета WB за 30 дней (обезличен: бренды,
названия, артикулы и суммы заменены на условные, все пропорции сохранены).
Уровень детализации: товар × склад — 15 350 строк.

## Что сделано
- **Python (pandas, numpy):** очистка данных, отсечение транзитных
  складов (сортировочные центры), свёртка, расчёт
  метрик — дни запаса, статус остатка, стоимость запаса
- **Power BI:** двухстраничный дашборд со срезами по бренду/категории/
  складу, условным форматированием, топ-листами

## Ключевые находки

**Концентрация выручки.** 16% товаров (393 из 2470) дают 80% всей
выручки. Основной ассортимент
кабинета держится на относительно узкой группе позиций.

**Здоровье запасов.** Из 2470 товаров: 1677 — в нормальном обороте,
1481 — неликвид (лежит без единой продажи хотя бы на одном складе),
1476 — распроданы подчистую при наличии спроса.

**Приоритеты на пополнение.** 89 товаров с заметным спросом
(>30 заказов за период) имеют запас меньше чем на 7 дней — это
конкретный список того, что нужно везти в первую очередь, чтобы
не потерять продажи.

**Неликвид.** 1481 позиция (48 672 штуки суммарно) не продаётся ни
на одном складе, где лежит. Это замороженные оборотные средства,
которые не движутся.

**Замороженный капитал.** Товары с медленной оборачиваемостью
(запаса хватит более чем на 90 дней при текущем темпе продаж)
держат основную часть стоимости складского запаса. Больше всего
такого капитала — на складах Коледино, Краснодар и Невинномысск.

**Структура бренда.** Топ-3 бренда дают больше половины выручки
кабинета — при работе с ассортиментом стоит учитывать эту
концентрацию.

## Ограничения данных
- Стоимость неликвида в деньгах не оценена: у товаров без единой
  продажи нет цены реализации, посчитать её не из чего. К API и раздела с ценами доступа нет.
- Остатки — снимок на момент выгрузки, не временной ряд; нет технической возможности выгрузить исторические данные;
  оборачиваемость оценена косвенно (остаток / средний спрос за период)
- Анализ не учитывает рекламные расходы и себестоимость —
  выводы касаются оборачиваемости и распределения, не прибыльности
